# MediSync_AI_Smart_Health_System
MediSync AI is a smart healthcare assistant that integrates AI-powered chatbot, prescription OCR, and patient health tracking. It helps users understand medical prescriptions, ask health-related questions, and manage their health records efficiently using Firebase and Gemini AI integration.
🧠 MediSync AI – Smart Healthcare Assistant

MediSync AI is an intelligent healthcare assistant web platform designed to help users with symptom guidance, prescription analysis, insurance prediction, health reminders, and AI-powered chatbot support. The system is built for accessibility, simplicity, and real-time health assistance.

🚀 Live Demo

🌐 Frontend: https://medisync-ai-110c3.web.app

🖥️ Backend: https://medisync-ai-backend.onrender.com

✨ Key Features
🤖 AI Health Chatbot
Real-time conversational AI assistant
Provides basic medical guidance (fever, headache, cold, etc.)
Smart fallback system for offline responses
API-powered response system (Gemini/OpenAI ready)
💊 Prescription Analyzer (OCR AI)
Upload or capture prescription images
Extracts and decodes handwritten text using OCR
Identifies medicines and suggests basic usage guidance
Camera support + file upload system
📊 Insurance Predictor (Smart Eligibility System)
Upload medical reports or insurance documents
Extracts patient data using OCR:
Patient name
Hospital name
Report/ID numbers (if present)
Suggests insurance schemes based on:
Age group
Disease condition
Job type (Government / Private)
Ration card type (White / Pink)
Displays relevant insurance application guidance links
⏰ Health Reminder System
Add medicine & health reminders
Select custom time, date, and sound
Firebase-based storage (real-time sync)
Notification-style popup alerts
Alarm-style sound notifications for medicine timing
Supports multiple reminder types (medicine, water, wellness)
🧠 Health Q&A System
Interactive health questionnaire
Dynamic questions instead of fixed loop
Generates personalized health suggestions
Lightweight AI-style logic system
