## 📌 Overview
MediSync AI is an intelligent web-based healthcare assistant that helps users with:
- Health-related chatbot assistance
- Prescription image decoding (OCR-based)
- Insurance eligibility prediction
- Health reminders with notifications
- Basic health Q&A system

The system is designed to simplify healthcare access using AI-inspired logic and automation.

---

## 🚀 Key Features

### 🤖 AI Health Chatbot
- Interactive chatbot for basic health queries
- Rule-based + AI-integrated responses
- Handles symptoms like fever, headache, cold, etc.

### 💊 Prescription Analyzer (OCR)
- Upload or capture prescription images
- Extracts and decodes medical text using OCR
- Identifies medicines and provides basic usage guidance

### 📊 Insurance Predictor
- Upload medical/insurance-related documents
- Extracts patient details like:
  - Name
  - Hospital name
  - ID / reference numbers (if present)
- Suggests suitable insurance schemes based on:
  - Age
  - Disease type
  - Eligibility factors (government/private schemes)
- Provides real-time application links (simulated + structured output)

### ⏰ Health Reminder System
- Add medicine or health reminders
- Set time, date, and sound alerts
- Stores data in Firebase Realtime Database
- Sends browser notification + sound alert at scheduled time

### 🩺 Health Q&A Module
- Simple multi-step health questionnaire
- Generates basic wellness suggestions based on responses

## ⚙️ How It Works

1. User uploads image or enters input
2. JavaScript processes data (OCR / logic rules)
3. Firebase stores reminders & user data
4. System generates instant health insights
5. Results are displayed in UI with recommendations
